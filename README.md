# thailand-landing-page

This is the promotion webpage to encourage people to revisit Thailand.

## Environment

Vue 3
Tailwind CSS
Vue 3 carousel

## Project Setup

```sh
npm install
```

### Move to folder

```sh
cd thailand-landing-page
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
